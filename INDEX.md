```
통합형 채팅 UI 디자인 ─┬─ 생성형 UI·리치 컴포넌트·응답 포맷   → CHI01~05 / DIS01·05·06·08
(텍스트+칩+UI+광고)    ├─ 통합 AI 채팅 사용자 행동(과의존 등)  → CHI06~08 / DIS09·11
                       └─ 커머스·설득·다크패턴(광고/넛지)       → CHI09·10 / DIS12
Orchestration/에이전트 인터페이스 ─ 하이브리드 GUI-LLM·의도·설명성 → DIS01~04·14 / CHI04
금융에서의 AI 사용자 행동 ───────── 투자/소비 의사결정 행동변화    → DIS13(MindStock) / CHI09(BuyMate)
```
